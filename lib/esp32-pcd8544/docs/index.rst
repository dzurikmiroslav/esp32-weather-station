===============================
The ESP32-PCD8544 documentation
===============================

This is the documentation for ESP32-PCD8544 (`esp32-pcd8544 <https://github.com/yanbe/esp32-pcd8544>`_).
ESP32-PCD8544 is a driver library implementation for PCD8544 LCD controller a.k.a.
`Nokia5110 LCD <https://www.google.co.jp/search?q=nokia5110+lcd&tbm=isch>`_.
